mwgadget.GuidedTour
===================

MediaWiki gadget that allows the creation of [Guided Tours](http://www.mediawiki.org/wiki/Guided_tours) using a variant of [Optimize.ly's Guiders](https://github.com/jeff-optimizely/Guiders-JS).
